``` ### **How to Run the New System** 1. **Place Files Together:** Make sure your new `index.html` file and your `rocket_server.py` file are in the **exact same directory** on your Raspberry Pi (e.g., `~/liamrocket`). 2. **Start the Server:** In a terminal on your Pi, navigate to that directory (`cd ~/liamrocket`) and run the server: ```bash python rocket_server.py ``` You should see messages that both the HTTP and WebSocket servers have started. 3. **Find Your Pi's IP Address:** In a new terminal, run `hostname -I` to get the IP address (e.g., `192.168.1.45`). 4. **Connect from Any Device:** Now, go to **any other computer or phone on the same Wi-Fi network**.
